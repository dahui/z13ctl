## z13ctl setup

Install udev rules to allow non-root access to ASUS RGB devices

### Synopsis

Writes /etc/udev/rules.d/99-z13ctl.rules granting members of a group read/write
access to the ASUS HID devices used for RGB control, then reloads and triggers
udev so the rules take effect immediately.

This command must be run with sudo.

```
z13ctl setup [flags]
```
